はじめてのプログラミング

最初のプログラムは最も基本的なものである、Hello,world!と出力されるプログラムを作ります。
出力するにはprint()を用います。文字列を出力したい場合、()の中にシングルクォーテーション、またはダブルクォーテーションで出力したい文字を囲みます。
画面右のプログラム入力エリアにprint('Hello,world!')と入力し、実行ボタンを押してみましょう。
※"プログラムを見る"をクリックすると答えが表示されます。プログラムがわからない時などに開いてみてください。

プログラムを見る 
print('Hello,world!')

四則演算

加算は+、減算は-、乗算は*、除算は/で行います。
6 + 4、8 - 7、6 * 2、9 / 3の計算をやってみましょう。printの()で式を囲んで実行してみましょう。

プログラムを見る 
print(6 + 4)
print(8 - 7)
print(6 * 2)
print(9 / 3)

割ったときの余りを求める場合は%累乗は**で求めます。
9を2で割ったときの余りと9の2乗を求めてみましょう。

プログラムを見る 
print(9 % 2)
print(9 ** 2)

変数とリスト

変数について

プログラミングに関する用語として「変数」があります。変数とは、数値や文字列などのデータを一時的に入れておく場所のことです。
変数には変数名として名前を付け、あとから参照したり値を変更したりすることができます。
変数の書き方 
変数名 = 数値、文字列など

変数名nameにあなたの名前を代入して出力してみましょう。

プログラムを見る 
name = 'あなたの名前'
print(name)

変数を使った計算ではx + 1やx * 1のような書き方以外にx += 1やx *= 1のように、短縮して書くことで見やすくなり、間違いを防ぐこともできます。変数名xに5を、変数名yに3を代入してxからyを減算してみましょう。

プログラムを見る 
x = 5
y = 3
x -= y
print(x)

リストについて

リストは複数の数値や文字列を1つの変数にまとめたものです。個々のデータを要素と言い、1つ1つの要素を,で区切って並べ[]で全体を囲んで作ります。
要素番号を指定することでリストの中から任意のデータを取り出したり、変更したりすることができます。
要素番号はリストの先頭から0で始まります。

リストの書き方 
変数名 = [数値や文字列,数値や文字列,数値や文字列]

変数名fruitsにバナナ、モモ、オレンジをリストに入れてバナナを取り出してみましょう。

プログラムを見る 
fruits = ["バナナ", "モモ","オレンジ"]
print(fruits[0])

リストに要素を追加したい場合は.append()を使います。

リストに要素を追加する 
変数名.append('追加したい要素')

先ほど作成したfruitsリストにあなたの好きなフルーツを追加してみましょう。

プログラムを見る 
fruits = ["バナナ", "モモ","オレンジ"]
fruits.append("イチゴ")
print(fruits)

for文

for文は、同じ処理を繰り返すときに用います。

for文の書き方 
for 変数名 in 文字列、リストなど:
    実行する処理

for文を使う際に、range()を用いることがよくあります。range()を使うと指定した回数分だけ処理が行われます。

range()の書き方 
for 変数名 in range(繰り返したい回数):
    実行する処理

range()を使って*を10個連続で出力するプログラム作ってみましょう。

プログラムを見る 
for i in range(10):
    print('*')

if文

if文は、条件に基づいて異なるコードを実行する際に使います。

if文の書き方 
if 条件:
    実行する処理

if文は、与えられた条件が満たされている行の箇所のコードを実行します。
tenに37を代入し、条件をtenが35以上の場合、猛暑日と出力されるプログラムを作ってみましょう。

プログラムを見る 
ten = 37
if ten >= 35:
    print('猛暑日')

条件が満たされなかったときに実行したい処理がある場合は、elseを用います。

elseの書き方 
if 条件:
    条件が満たされているときに実行する処理
else:
    条件が満たされていないときに実行する処理

ageに20を代入して、18以下なら未成年、そうでなければ成人と出力されるプログラムを作りましょう。
実行することができたらageに他の数字も入れて実行してみましょう。

プログラムを見る 
age = 20
if age < 18:
    print('未成年')
else:
    print('成人')

条件が満たされなかったときに、さらに別の条件で分岐させたいときはelifを用います。

elifの書き方 
if 条件:
    条件が満たされているときに実行する処理
elif 条件:
    elifの条件で満たされたときに実行する処理
else:
    条件が満たされていないときに実行する処理

変数名btに血液型を設定し、btを判定して血液型を出力するプログラムを作りましょう。

プログラムを見る 
bt = 'O'
if bt == 'A':
    print('血液型はA型です')
elif bt == 'B':
    print('血液型はB型です')
elif bt == 'O':
    print('血液型はO型です')
else:
    print('血液型はAB型です')
